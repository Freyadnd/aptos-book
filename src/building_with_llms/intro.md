# Building with LLMs

Large language models (LLMs) are becoming a standard part of the developer toolkit.  Whether you're writing Move contracts, reviewing code for security issues, or building autonomous agents that transact on-chain, LLMs can meaningfully accelerate your workflow on Aptos.

This section covers practical ways to use LLMs alongside Aptos development:

- **[Writing Move with LLMs](writing_move_with_llms.md):** Prompting strategies and patterns for generating correct Move code.
- **[Auditing with LLMs](auditing_with_llms.md):** Using LLMs as a first-pass security review tool for your contracts.
- **[AI Agents on Aptos](ai_agents_on_aptos.md):** Building autonomous agents that read from and write to the Aptos blockchain.
- **[llms.txt and Aptos](llms_txt_and_aptos.md):** How the Aptos ecosystem exposes documentation in LLM-friendly formats.

> LLMs are a productivity multiplier, not a replacement for understanding the underlying platform.  The chapters that follow assume you have worked through earlier sections of this book and are comfortable with Move fundamentals.
